Lesson 9: Semantic HTML & Accessibility

Semantics, Alt Text, Roles, ARIA Basics

Semantic Example

<header>
  <h1>Website Title</h1>
</header>

<nav>
  <ul>
    <li><a href="#">Home</a></li>
  </ul>
</nav>

Semantic HTML gives meaning to content. Accessibility features like alt text and ARIA roles help users with disabilities navigate pages.
