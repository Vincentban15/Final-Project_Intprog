What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

HTML is the backbone of all web pages. It structures content and defines elements like headings, paragraphs, links, images, and more.
