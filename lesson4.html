Lesson 4: Links & Navigation

Anchors, Internal/External Links, Mailto, Link Attributes

Creating Links

Use the <a> tag to create hyperlinks:

<a href="https://www.example.com">Visit Example</a>
<a href="#section2">Go to Section 2</a>

Anchors allow users to navigate within a page or to external sites. Attributes like target="_blank" open links in new tabs.
