Lesson 11: Meta & Document Head

Doctype, html/head/body, Charset, Viewport, Title, Metadata

Document Head Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Page</title>
</head>
<body>
  <h1>Hello</h1>
</body>
</html>

The document head contains metadata about the page, including title, character set, and viewport settings for responsive design.
