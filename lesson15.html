Lesson 15: Project / Capstone

Build a Simple Static Webpage Using All Learned Topics

Project Example

<!DOCTYPE html>
<html>
<head>
  <title>My Webpage</title>
</head>
<body>
  <header><h1>Welcome</h1></header>
  <nav><a href="#">Home</a></nav>
  <section><p>Page content here.</p></section>
  <footer><p>Footer info</p></footer>
</body>
</html>

This final project combines headings, paragraphs, links, images, lists, forms, semantic layout, multimedia, and best practices into a complete static webpage.
